Memory Task Instructions

The following task is designed to measure your working memory capacity.

You will see a series of trials. In each trial, you will be shown:

  1. A simple math problem (e.g., (2*3)-1 = 5?). You must decide if it is True or False.
  2. A single letter, which will be displayed for one second. Your goal is to remember these letters in the order they appear.

After a set of these pairs, you will be asked to recall the letters from that set in their original order.

It is important to try your best on both the math problems and on remembering the letters. Please focus and try to maintain a balance between both tasks.
